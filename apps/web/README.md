# Mala Min 360 Web

Next.js frontend foundation for Mala Min 360.

## Commands

```powershell
copy .env.example .env.local
npm install
npm run dev
npm run build
```

The backend API must be running at the URL in `.env.local`.

Default development API URL:

```text
http://localhost:5000
```

Local URL:

```text
http://localhost:3000
```

Current routes:
- `/`
- `/login`
- `/dashboard`
- `/properties`
- `/a/demo-agency/demo-property`
